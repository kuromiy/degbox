# イラスト詳細画面 仕様書

## 1. 概要
イラストの詳細情報を表示し、タグによる検索への導線を提供する画面。

## 2. UI構成

画面は上から順に以下の要素で構成される：

```
┌─────────────────────────────┐
│        イラスト画像         │
│                             │
└─────────────────────────────┘
┌─────────────────────────────┐
│          タグ一覧           │
│  [タグ1] [タグ2] [タグ3]   │
└─────────────────────────────┘
┌─────────────────────────────┐
│        画像情報             │
│  - ファイル名               │
│  - サイズ                   │
│  - 登録日時                 │
│  など                       │
└─────────────────────────────┘
```

### 2.1 イラスト画像
- イラストを表示
- 画面サイズに応じて適切にリサイズ表示

### 2.2 タグ一覧
- イラストに紐づくタグをボタン/チップ形式で表示
- 各タグはクリック可能

### 2.3 画像情報
- ファイル名
- ファイルサイズ
- 登録日時
- その他メタデータ

## 3. 機能要件

### 3.1 イラスト表示
- イラストIDに基づいて該当イラストを取得・表示
- 画像が存在しない場合はエラー表示

### 3.2 タグ機能
- **クリック時の動作**: タグをクリックすると、そのタグで絞り込んだ検索画面へ遷移
- タグが存在しない場合は「タグなし」と表示

### 3.3 画像情報表示
- イラストに紐づくメタデータを読み取り専用で表示

## 4. 画面遷移

### 4.1 タグクリック
- **遷移先**: 検索画面（イラスト一覧画面）
- **パラメータ**: 選択されたタグで検索条件を設定
- **動作**: クリックされたタグでフィルタリングされたイラスト一覧を表示

## 5. 将来実装予定

以下の機能は将来的に実装予定：

### 5.1 編集機能
- **UI配置**: 現時点でボタンのみ配置
- **ロジック**: 後日実装
- **想定機能**:
  - タグの追加・削除
  - 画像情報の編集

### 5.2 削除機能
- **UI配置**: 現時点でボタンのみ配置
- **ロジック**: 後日実装
- **想定機能**:
  - イラストの削除
  - 削除前の確認ダイアログ表示

## 6. 技術的検討事項

### 6.1 コンポーネント構成（例）
```
IllustDetail/
  ├─ IllustImage.tsx        # イラスト画像表示
  ├─ TagList.tsx            # タグ一覧
  ├─ IllustInfo.tsx         # 画像情報表示
  └─ EditDeleteButtons.tsx  # 編集・削除ボタン（将来実装）
```

### 6.2 データ取得
- イラストIDをパラメータとして受け取る
- APIまたはデータストアからイラスト情報を取得

### 6.3 状態管理
- イラストデータ
- ローディング状態
- エラー状態

## 7. 備考
- 編集・削除機能のUIは配置するが、クリック時は「準備中」等のメッセージを表示
- タグクリック時の検索画面への遷移は初期実装に含める
